Installation

Pour windows : https://git-for-windows.github.io
Télécharger le fichier : Git-2.14.2.3-64-bit.exe et Installer sur C / Programme Files

Attention ne pas faire Next sur l'étape "Ajusting your path environment" et choisir la 3eme option :

Commandes GIT

Les "_" sont des espaces

Pour connecter la console :

git_config_--global_user.email_"emailgithub@gmail"

git_config_--global_user.name_"usernamegithub"Pour connecter mon ordi avec mon compte GITHUB

git_status_Pour lister l'etat

git_add_•Ajoute tous les fichiers to commit

git_add_.gitignore_README.mdAjoute ces 2 fichiers to commit

git_commit_-m_"initial_commit"Premier commit du projet

git_push_origin_masterPour lancer le commit du master

Pour créer une branche :

git_branch_nom-brancheCréer une branche, la nommer, y aller.

git_branch_Lister les branches créées

Pour supprimer une branche :

git_branch_-D_nom-brancheLa suppression sera visible sur GITHUB seulement après le prochain PUSH

git_checkout_nom-branchePour sortir de la branche

A faire avant tout commit :

git_pull_origin_masterSynchro avec Github

Recupérer un projet depuis Github :

git_clone_URL recupérée sur GithubProjet de Github en Local

Pour revenir à une version antérieure :

git_log_QUI a fait QUOI et QUAND

git_log_-n1Premier commit

METHODE 1 :

git_reflog_-n2Donne le n° second commit : ici par exemple 4994164

git_reset_--hard_4994164Indique le n° de commit pour le reset

git_push_--forceEffectue le Reset précedement demandé

METHODE 2 :

git_revert_--hard_4994164Indique le n° de commit pour le revert

git_push_origin_masterPour lancer le commit du master
