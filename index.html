INITIATION GITHUB

Installation

Pour windows : https://git-for-windows.github.io
Télécharger le fichier : Git-2.14.2.3-64-bit.exe et Installer sur C / Programme Files

Attention ne pas faire Next sur l'étape "Ajusting your path environment" et choisir la 3eme option :

Commandes

Les "_" sont des espaces

git_config_--global_user.email_"monemailpourgithub@gmail"

git_config_--global_user.name_"monusernamepourgithub"Pour connecter mon ordi avec mon compte GITHUB

git_status_Pour lister l'etat

git_add_•Ajoute tous les fichiers to commit

git_add_.gitignore_README.mdAjoute ces 2 fichiers dans la liste des fichiers to commit

git_commit_-m_"initial_commit"Pour le premier commit du projet

git_push_origin_masterPour lancer le commit du master

git_branch_nom-du-fichierPour créer une branche et lister celle déjà créées

git_branch_nom-branchePour créer une branche et lister celle déjà créées

git_checkout_nom-branchePour sortir de la branche

A faire avant tout commit :

git_pull_origin_masterPour recharger le projet en local depuis Github

Recupérer un projet depuis Github :

git_clone_URL recupérée sur GithubPour copier un projet de Github en Local

Pour revenir à une version antérieure :

git_log_Permet de visualiser tous les commit en détail, QUI a fait QUOI et QUAND

git_log_-n1Permet de visualiser le premier commit

METHODE 1 :

git_reflog_-n2Donne le n° de commit du second commit : ici par exemple 4994164

git_reset_--hard_4994164Indique le n° de commit pour le reset

git_push_--forceEffectue le Reset précedement demandé

METHODE 2 :

git_revert_--hard_4994164Indique le n° de commit pour le revert

git_push_origin_masterPour lancer le commit du master
